fn convert_to_rotation_segment(degrees: f32) -> u8
Expand description

Converts a rotation in degrees to a 16-segment rotation value (0-15).

This is equivalent to vanilla’s RotationSegment.convertToSegment(float). Each segment is 22.5 degrees, and rotation is measured clockwise from south.
