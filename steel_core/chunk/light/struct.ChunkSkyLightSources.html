Struct ChunkSkyLightSources

pub struct ChunkSkyLightSources {
    min_y: i32,
    pub(super) heightmap: [i32; 256],
}
Expand description

Per-chunk cache of the lowest skylight source edge in each X/Z column.

Vanilla stores this in a 256-entry SimpleBitStorage. Steel keeps absolute i32 Y values instead; the cached semantics are the same, and this avoids a new bit-storage abstraction before another system needs one.
