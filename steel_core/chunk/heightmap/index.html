Heightmap implementation for tracking the highest blocks in a chunk.

Heightmaps are used for various purposes like spawning, pathfinding, and rendering.

During worldgen, ProtoHeightmaps stores a dynamic set of heightmaps (worldgen types before CARVERS, final types after). When a proto chunk is promoted to a full LevelChunk, the final heightmaps are moved directly into ChunkHeightmaps via ChunkHeightmaps::from_proto.
