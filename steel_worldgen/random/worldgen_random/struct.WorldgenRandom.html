Expand description

Vanilla’s WorldgenRandom when constructed for biome decoration.

Feature decoration always constructs WorldgenRandom(new XoroshiroRandomSource(...)). Sampling then goes through BitRandomSource.next*, so it does not match raw XoroshiroRandomSource for nextInt, bounded ints, doubles, longs, or gaussians.
