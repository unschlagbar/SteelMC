Struct Aabb 

Source 
pub struct Aabb<T, I> {
    min: T,
    max: T,
    p: PhantomData<I>,
}
Expand description

Generic axis-aligned bounding box.

T is the vector type (e.g. DVec3 or IVec3) and I is a marker that differentiates coordinate spaces (block-local, world, structure).
